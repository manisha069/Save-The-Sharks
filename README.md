# Save-The-Sharks
## Overview

This is a basic AndroidStudio project made in JAVA. The basic function of the project is to share with it's users an interesting fact about sharks and then forward them on the click of a button to Save the Ocean website if they want to contribute and donate to the cleaning up of the Oceans.
